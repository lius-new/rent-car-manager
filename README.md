### 租车系统(前端)


前端采用vue3+vite+vue-router+axios+pinia+naive-ui+sass等相关技术。

### 目录结构
```
src/
├─App.vue
├─main.ts
├─utils.ts   # 工具模块
├─styles     # 样式
├─stores     # store
├─serve      # API
├─router     # 路由
├─pages      # 页面组件
├─mock       # 模拟数据   
├─components # 公共组件
├─assets
```



### 项目运行

确保你已经安装了nodejs

```
npm install

npm run dev
```
